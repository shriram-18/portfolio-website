# Shriram Patil - Portfolio Website

A clean, responsive, and production-ready full-stack personal portfolio demonstrating my skills, backend projects, and contact information.

## 🚀 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Glassmorphic UI
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Security & Utilities:** Helmet, CORS, Rate Limiting, Nodemailer

## 📁 Project Structure
- **/frontend:** Contains the semantic HTML, CSS styling, and client-side data fetching logic.
- **/backend:** Contains the Express server, REST API routes, controllers, and MongoDB models.
